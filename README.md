# 🎨 Background Color Changer (Mini Project 1)

## 📖 Description
This is a simple Background Color Changer web application built using HTML, CSS, and JavaScript. Users can change the webpage background color by selecting different colors. This project demonstrates DOM manipulation, event handling, and responsive UI design.

## ✨ Features
- Change background colors instantly
- Simple and user-friendly interface
- Responsive design
- Beginner-friendly project

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript

